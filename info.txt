 - senha (senha deve ter extamente 6 digitos)
 - saldo

Consultar saldo

Depositar 
 - o sistema ira conferir se o id e senha existem caso seja true irá pedir o valor de deposito

Sacar 
 - o sistema ira conferir se o id e senha existem caso seja true irá pedir o valor de saque

Transferencia

 o mesmo ira pedir o valor que deseja transferir

Pagina inicial
